# Cylinder

Cylinder是一个个人兴趣开源项目，目标是提供一个帮助开发者和研究人员评估和比较不同大模型在编程任务上的表现，不具有绝对参考意义。


⚠ 这个项目刚刚起步 ⚠


## 如何帮助/加入我们？
Cylinder 的目标是提供一个全面的平台，帮助开发者和研究人员评估和比较不同大模型在编程任务上的表现。
可以通过提交 issue 或者 pull request 来帮助我们。我们会不定期的在Project发布目标和帮助请求。

为了确保一致性，一切提交代码必须遵守以下原则

- 项目结构
  - 将功能按照分类构建文件夹，便于维护和扩展。
  - 使用标准的文件命名规范，便于查找和定位。
  - 文件夹下必须附带README.md文件，用于描述该文件夹的功能和使用方法。

- python
  - 必须经过ruff check代码审查
  - 必须声明类型类型
  - 函数必须注释其作用
